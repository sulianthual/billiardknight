Instructions

1.) This folder handles compiling of python code into an executable using pyinstaller.
    If on windows, have also 7zip installed

2.) using terminal (or cmd in windows), go to within this folder

3.) edit script: the appnamepre will determine the output zip folder name

4.) launch script. If linux/mac: ./compile_linux and if windows: compile_win10

5.) follow prompt (answer Yes if needed)

6.) the code runs pyinstaller, then cleans up. 

7.) the output zip file is found along the main game folder
